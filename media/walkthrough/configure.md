# Configure your workflow

The configuration wizard lets you customize how claudeloop runs in your project.

## Options

- **Model** -- choose which AI model to use
- **Verify** -- run verification after each phase
- **Refactor** -- run refactoring after each phase
- **Dry run** -- preview the plan without executing

Settings are stored in `.claudeloop/.claudeloop.conf` inside your workspace.

Click **Open Configuration Wizard** to get started.
