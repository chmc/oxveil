# Create your first plan

A `PLAN.md` file defines the work claudeloop will execute, broken into numbered phases.

## Plan structure

Each phase has a number, title, and description:

```markdown
## Phase 1: Set up project

Create the initial project structure with package.json and src/ directory.

## Phase 2: Implement core logic

Build the main processing pipeline.
```

Claudeloop executes phases sequentially, committing progress after each one.

Click **Create PLAN.md** to scaffold a starter plan in your workspace.
