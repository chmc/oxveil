# Run your first session

With a plan in place, you can start a claudeloop session.

## What to expect

1. Claudeloop reads your `PLAN.md` and begins at the first incomplete phase
2. The Oxveil sidebar shows real-time progress
3. The status bar displays the current phase and elapsed time
4. Each completed phase is committed to git automatically

## Monitoring

- Click a phase in the sidebar to view its log
- Use **Oxveil: Show Dependency Graph** to visualize phase relationships
- Open **Oxveil: Open Replay** after a run to review the full session

Click **Start claudeloop** to begin.
