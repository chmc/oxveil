# Detect claudeloop

Oxveil is a VS Code frontend for **claudeloop**, a CLI tool that orchestrates multi-phase AI coding sessions.

Before you can use Oxveil, claudeloop must be installed and available on your PATH.

## What happens

- Oxveil checks for claudeloop on activation
- The status bar shows whether claudeloop was found
- If missing, click **Install claudeloop** to open an installation terminal

Once detected, the Oxveil sidebar and commands become available.
