Oxveil v0.4

Multi-Root Workspace — folder-scoped sessions with grouped tree view, contextual status bar, and folder picker

Phase Tree View — Folder Grouping

When multiple workspace folders are open, phases are grouped under collapsible folder nodes. Each folder shows its session status. Single-root workspaces show the flat list (unchanged from v0.1).

Status Bar — Folder Context

Multi-root adds a folder name prefix and a summary of other roots. Single-root status bar is unchanged from v0.1. Click focuses the phase tree view.

Shown when a command needs a workspace folder in multi-root context. Skipped for single-root or when folder is implicit (e.g., triggered from tree view context menu). Two-line items with session status.
